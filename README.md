# file-operation-using-go
Task 1: Extract files from .gz format to multiple csv files.


# Steps
Step 1: Download & Install go from https://go.dev/doc/install

Step 2: Initialize a new Go module
```bash
go mod init file-operation-using-go
```

Step 3: Create a new Go file named main.go and add hello world code
```go
package main
import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```
Step 4: Run the Go program
```bash
go run main.go
```
